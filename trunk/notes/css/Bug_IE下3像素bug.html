北京时间6月13日（周日）凌晨2点30分，南非世界杯C组第一场比赛在勒腾斯堡皇家巴弗肯球场进行，交战双方是英格兰和美国，上半场比赛杰拉德开场4分钟就接到赫斯基助攻破门，而半场结束前格林低级失误送礼，邓普西远射为美国队扳成1-1平，最终双方各得一分。【
