Git参考笔记

1、创建验证用的公钥

  • 1)运行Git Bash
  • 2)在弹出的dos命令框中使用命令：

    ssh-keygen -C 'you email louxiaojian@gmail.com' -t rsa

    然后会询问一些保存文件的位置，设置密码神马的，直接回车，回车，回车，默认的就可以了, 就会为你生成一个 SSH Key。 因为主要是在本地使用，可不使用口令，直接回车就可以了！

2、上传公钥

在 github.com 的界面中 选择右上角的 Account Settings，然后选择 SSH Public Keys ，选择新加。 Title 可以随便命名，Key 的内容拷贝自 ~/.ssh/id_rsa.pub 中的内容，完成后，可以再使用 ssh -v git@github.com 进行测试。看到下面的信息表示验证成功。
