关于我们

中国派康网―PET/CT信息服务网隶属上海派领康信息技术有限公司，中国派康网―PET/CT信息服务网是一个融专业性、功能性于一体的网络预约、网络宣传、技术推广、PET/CT信息服务的一个公益性、专业性门户网站。 中国派康网充分将现代信息技术应用到PET/CT医疗技术的服务中，通过普及PET/CT健康意识理念，帮组受检者改善预约服务流程，提高服务质量，建立一套免费、完善、周密、个性化的以PET/CT检查为核心的高端体检的预约服务，提供更方便、快捷的服务流程，宣传优质健康信息理念，创建和谐健康环境，不断提高高端客户群众健康水平，促进社会和谐。
