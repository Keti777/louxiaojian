Git参考笔记

1、创建验证用的公钥

  • 1)进入到git安装目录，运行Git Bash
  • 2)运行Git Bash后弹出一个dos命令框，在框中使用命令：ssh-keygen -C 'you email louxiaojian@gmail.com' -t rsa 会在用户目录 ~/.ssh/ 下建立相应的密钥文件 可以使用 ssh -v git@github.com 命令来测试链接是否畅通

2、上传公钥

在 github.com 的界面中 选择右上角的 Account Settings，然后选择 SSH Public Keys ，选择新加。 Title 可以随便命名，Key 的内容拷贝自 ~/.ssh/id_rsa.pub 中的内容，完成后，可以再使用 ssh -v git@github.com 进行测试。看到下面的信息表示验证成功。
